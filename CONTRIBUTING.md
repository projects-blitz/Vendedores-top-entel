Contributing
============

Gracias por interesarte en mejorar este proyecto.

Guía rápida:

- Ejecutar localmente:
  1. npm install
  2. npm run dev
- Branches: usa feature/<nombre> para nuevas funcionalidades.
- Haz PR hacia main; incluye descripción y screenshots.
- Mantén commits pequeños y con mensajes claros.

Testing & CI:
- Un workflow CI construye el proyecto en pushes y PRs a main.

Código:
- Mantener lógica en src/utils y componentes presentacionales en src/components.
